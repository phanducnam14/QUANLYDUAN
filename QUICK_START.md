# ⚡ Quick Start (Chạy nhanh trong 5 phút)

## 🎯 Các yêu cầu tối thiểu

- Java 17+
- Node.js & npm
- MongoDB (đang chạy)

## 🚀 Chạy ngay

### Terminal 1: Backend
```bash
./mvnw spring-boot:run
# Đợi đến khi thấy "Started CoreSystemApplication"
```

### Terminal 2: Frontend
```bash
cd client-app
npm install  # Lần đầu tiên
npm run dev
# Mở http://localhost:5173
```
